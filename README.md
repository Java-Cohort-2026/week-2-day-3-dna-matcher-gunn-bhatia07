# W2D3 Assignment: The DNA Sequence Matcher

### Your Mission
Write two methods outside of your `main` block:
1. `buildSequence(String[] frags)`: Loop through the array and use a `StringBuilder` to append every piece together. Return the final string.
2. `findMutation(String[] samples, String target)`: Use a Linear Search algorithm to find the target. Return the index position. If it doesn't exist, return `-1`.

*Warning: If you use `==` instead of `.equals()` to compare the DNA strings, your search will fail!*
